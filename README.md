# Urban Style

Aluno: Luiz Gabriel Abreu Oliveira

Este projeto é um site fictício de uma loja de roupas streetwear, desenvolvido utilizando HTML.

O site possui 3 páginas:
- Página inicial
- Página de produtos
- Página de contato

O objetivo da atividade é praticar a estrutura básica do HTML, uso de imagens, links e navegação entre páginas.
